# 英文読解

## 英文読解の使用記号表

| 記号 | 意味 |
| --- | --- |
| S | 主語 |
| V | 動詞 |
| O | 目的語 |
| C | 補語 |
| M | 修飾語 |
| Ving | 現在分詞 |
| p.p. | 過去分詞 |
| rel. | 関係詞 |
| prep. | 前置詞 |
| adv. | 副詞 |
| conj. | 接続詞 |


## 読解ポイント
- TOFLE ibtのテスト対策を意識した読解を行う
- 和訳をつける
- 各単語の意味をつける
  - 添付している文章にわからない単語に強調表示「`強調表示`」しているため、その単語の意味をまとめる
  - TOFLE ibt受験において重要と思われる単語も追記する
- それぞれを読解した際、どのように文章が構成されているのか解説を加える
  - 読み方のポイント等
- 修正したファイルの先頭に、英文読解で使用している記号の意味も参照できるようにしたい
