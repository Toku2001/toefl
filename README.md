# TOEFL 学習方針

## 進め方
- 1.単語学習(本試験レートは飛ばす)
  - わからない単語をまとめる
    - テキストを一周する(GWまで)
    - 単語学習とリスニング(5月末まで)
    - 過去問題を解く(6月末まで)


- 2.シャドーイング
  - 音で聞き取れるようにする
- 3.模擬問題を解く


##　試験受験日
- 2026年6月末　(50-60点台)
- 2026年9月末　(70-80点台)
- 2026年12月　(90点台)⭕️
