# 単語リスト作成ルール

## フォーマット

各単語は `<details>`/`<summary>` タグで囲み、クリックすると詳細が展開される形式にする。

```markdown
<details>
<summary>単語</summary>

**意味:** ...

**基本形:** ...（該当する場合のみ）

**アクセント:** ...

**類義語:** ...

</details>
```

## 各項目のルール

### 意味
- 日本語で記載する
- 品詞が複数ある場合はセミコロン（；）で区切る
- 例：`名詞の意味；動詞の意味`

### 基本形
- 見出し語が複数形・三人称単数形・過去形などの場合のみ記載する
- 動詞と名詞で形が異なる場合はそれぞれ明記する
- 例：`remain（動詞）/ remains（名詞・複数形）`

### アクセント
- IPA記号は使わず、辞書式発音表記（`kräp` スタイル）で記載する
- 強勢（第一アクセント）のある音節を**太字の大文字**で示す
- 見出し語が基本形でない場合は、**基本形のアクセント**を記載する
- 主な発音記号の凡例：

| 記号 | 音のイメージ |
|---|---|
| ā | 長いエイ音（face） |
| ä | 短いア音（cat） |
| ē | 長いイー音（see） |
| ə | 曖昧な短母音（schwa） |
| ī | 長いアイ音（hi） |
| ĭ | 短いイ音（bit） |
| ō | 長いオ音（go） |
| ô | 丸みのある短オ音（law） |
| ū | 長いウ音（food） |
| **大文字** | 強勢のある音節 |

### 類義語
- 英語で3語程度記載する
- カンマ区切りで並べる

### 備考
- 記入されている単語の順番は変えないでください
- 記入されている単語を削除しないでください
- 同じ単語が含まれていても、フレーズが重複していなければ削除しないで下さい
